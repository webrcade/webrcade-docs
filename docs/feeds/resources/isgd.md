# is.gd

is.gd is a URL shortening service that can be used to reduce the length of URLs in your webЯcade feeds.

## How to Use is.gd

1. Go to [is.gd](https://is.gd/).
2. Paste your long URL into the input box.
3. Click the `"Shorten!"` button.
4. Copy the resulting short URL and use it in your feed.

## Aliased URLs {#aliased-urls}

is.gd allows you to create custom, human-readable (aliased) URLs that point to your original, longer URLs. This is useful for making feed URLs easier to remember and share.

### How to Create an Aliased URL

1. Go to [is.gd](https://is.gd/).
2. Paste your long URL (such as a Pastebin feed URL) into the input box.
3. Click on `"Further options/custom URL"` to expand the options.
4. Enter your desired alias in the "Custom short URL" field (e.g., `tutorialfeed`).
5. Click the `"Shorten!"` button.
6. If the alias is available, you will receive a short URL like `https://is.gd/tutorialfeed`.

### Example

Original URL:
```
https://pastebin.com/raw/Fchkqw6g
```
Aliased with is.gd:
```
https://is.gd/tutorialfeed
```

You can now use the aliased URL in your webЯcade feed or share it with others for easy access.

## Notes
- You can optionally customize the short link using the "Further options/custom URL" feature on the is.gd site.
